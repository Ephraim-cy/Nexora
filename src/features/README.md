# Nexora Features Directory

This directory contains the feature-specific modules of Nexora:

- `capture/`: Screens and components for capturing screenshots, photos, scanning documents, and processing them.
- `search/`: Full-text and semantic search UI and search logic.
- `gallery/`: Smart gallery cleaner, duplicate detection, and image grouping interface.
- `study/`: Whiteboard scanner, handwriting recognition, flashcards, and spaced repetition engine.
- `finance/`: Receipt and expense tracking dashboards, reports, and category visualizers.
- `paywall/`: Gated features, subscription settings, and RevenueCat integration.
- `settings/`: App configuration, user profiles, privacy toggles, and data export.
