# Nexora Core Systems Directory

This directory contains the underlying core systems and utilities:

- `ai/`: Content classification, text recognition (OCR), image analysis fallbacks, and local ML configurations.
- `database/`: Database clients, query configurations, schemas, and migrations.
- `security/`: Key storage, local biometric authentication, file encryption (AES-256), and cryptographic helpers.
- `sync/`: Sync queues, offline-first CoreData synchronization logic, and network connection listeners.
